
INTERFACES : 

- It is just like class but with some fixed properties
- It consists of pure absctration
- It cannot have constructors 

Properties of Interfaces
1. All the fields in interfaces are public,static and final by default.
2. All methods are public & absctract by default
3. A class that implements and interface must implement all the methods declared in the interface.
4.Interfaces support the functionality of mutiple inheritence.
